# 1. Getting Started: The Initial Steps

### 1.1 Installation & Dependencies
CineFlow Suite is a self-contained macOS application designed to run on **Apple Silicon (M1/M2/M3)** for maximum performance.
- **Native Bridges:** The app includes professional-grade sidecars for **BRAW (Blackmagic RAW)** and **REDlink (RED R3D)** processing. No external installation is required.
- **FFmpeg Integration:** Advanced media analysis is handled by an internal, optimized FFmpeg engine.

### 1.2 Your First Project
1. **Initialize:** Open the app and select your root media folder.
2. **Scan:** Use the **Contact Sheet** or **Review Core** to index your footage.
3. **Verify:** Use **Safe Copy** if you are just offloading media to ensure 1:1 data integrity.

# 3. The Utilities Suite (Micro-Apps)
*Small tools, massive impact.*

The CineFlow Utilities are designed to solve common set-side friction points instantly.

### 💎 Featured Highlights

- **Safe Copy (VCS):** 
    - **Benefit:** Not just a copy tool; a verification system. It performs byte-by-byte hash comparisons between source and destination.
    - **Pain Killed:** The fear of a corrupted card or a partial copy.
- **Mosaic Builder:** 
    - **Capability:** Generates stunning, high-density contact sheets (up to 400+ frames per page).
    - **Usage:** Perfect for "Wrap Reports" or visual indexing of massive projects.
- **Duplicate Finder:** 
    - **Benefit:** Scans petabytes of data locally to find byte-identical copies of media files, freeing up expensive RAID storage.
- **Standardized Folder Creator:** 
    - **Capability:** One-click generation of professional Post-Production folder structures (e.g., 01_FOOTAGE, 02_AUDIO, 03_ASSETS).

### 🧮 The Calculator Suite
- **Crop Factor Calc:** Mutant-accurate focal length conversion for multi-camera setups.
- **Transfer Time Calc:** Predicting wrap times based on data volume and drive speed (Thunderbolt 3/4 vs USB-C).
- **File Size Calc:** Estimating storage requirements for RAW, ProRes, and H.264 formats.

# 4. Technical Requirements & Security

- **Sandboxing:** CineFlow operates within the strict macOS App Store sandbox. It uses **Security-Scoped Bookmarks** to maintain access to your media libraries without compromising system security.
- **Privacy:** **Zero-Cloud Dependency.** Your footage, your metadata, and your project logic never leave your local machine or external storage.
- **Hardened Runtime:** The application is built with Apple’s Hardened Runtime to prevent tampering and malicious code injection.
